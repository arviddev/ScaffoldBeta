# Scaffold – Beta

Ticket- und Kalender-App für macOS (14+).

## Download

**[⬇️ Scaffold 0.1 (.dmg)](../../raw/main/Scaffold-0.1.dmg)**

### Installation
1. DMG herunterladen und öffnen
2. `Scaffold.app` in den Ordner **Programme** ziehen
3. App starten

Die App ist mit einer Apple Developer ID signiert und von Apple notarisiert –
sie läuft auf jedem Mac ohne Sicherheitswarnung.
