# Scaffold – Beta

Ticket- und Kalender-App für macOS (14+).

## Download

**[⬇️ Scaffold 0.1 (.dmg)](../../raw/main/Scaffold-0.1.dmg)**

### Installation
1. DMG herunterladen und öffnen
2. `Scaffold.app` in den Ordner **Programme** ziehen
3. App starten

Die App ist mit meiner Apple Developer ID signiert und von Apple notarisiert,
sie läuft auf jedem Mac mit macOS 14+ unbedenklich (falls alles nach Plan läuft)

Es handelt sich um einen Pre-Release, die App befindet sich in der Testphase, alle Grundfunktionen sind da. 

In Planung/Entwicklung befinden sich derzeit:
  - weitere Features (zum Beispiel Integration von Claude in den Kalender)
  - weitere Versionen für folgende Betriebssysteme: Linux, Windows, Android, iOS, iPadOS
